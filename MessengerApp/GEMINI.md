# 사내 메신저 (MessengerApp)

## 프로젝트 개요
이 프로젝트는 Rust 언어와 Egui 프레임워크를 사용하여 개발된 간단한 사내 메신저 애플리케이션입니다. 사용자들은 채팅방에서 메시지를 주고받을 수 있으며, 기본적인 사용자 인터페이스가 구현되어 있습니다.

## 주요 기능
*   **채팅 메시지 전송**: 사용자가 메시지를 입력하고 전송할 수 있습니다. 엔터 키 또는 '전송' 버튼을 통해 메시지를 보낼 수 있습니다.
*   **채팅 기록 표시**: 전송된 메시지들은 중앙 패널에 시간 순서대로 표시됩니다.
*   **사용자 목록 (더미 데이터)**: 왼쪽 사이드바에 현재 접속 중인 사용자 목록을 표시합니다 (현재는 더미 데이터).
*   **반응형 UI**: Egui 프레임워크를 활용하여 직관적인 사용자 인터페이스를 제공합니다.

## 사용 기술
*   **Rust**: 안정성, 성능 및 신뢰성을 강조하는 프로그래밍 언어입니다.
*   **Eframe**: Egui 애플리케이션을 빌드하기 위한 프레임워크입니다.
*   **Egui**: Rust로 작성된 즉시 모드(immediate mode) GUI 라이브러리입니다.

## 프로젝트 실행 방법

1.  **Rust 설치**: 시스템에 Rust가 설치되어 있지 않다면, [Rust 공식 웹사이트](https://www.rust-lang.org/tools/install)에서 설치 가이드에 따라 Rust를 설치합니다.
2.  **프로젝트 클론**: 이 저장소를 로컬 컴퓨터로 클론합니다.
    ```bash
    git clone [저장소 URL]
    cd MessengerApp
    ```
3.  **애플리케이션 실행 (개발 모드)**: 프로젝트 디렉토리에서 다음 명령어를 실행하여 애플리케이션을 빌드하고 실행합니다.
    ```bash
    cargo run
    ```
4.  **Windows 실행 파일 생성**: 윈도우즈 11에서 실행 가능한 파일을 생성하려면 다음 명령어를 사용합니다.
    ```bash
    cargo build --release --target x86_64-pc-windows-gnu
    ```
    생성된 실행 파일은 `target/x86_64-pc-windows-gnu/release/MessengerApp.exe` 경로에서 찾을 수 있습니다.

## 코드 구조
*   `src/main.rs`: 애플리케이션의 메인 로직을 포함합니다. Eframe 애플리케이션의 진입점과 `MessengerApp` 구조체, 그리고 `eframe::App` 트레이트 구현이 포함되어 있습니다.
*   `Cargo.toml`: Rust 프로젝트의 의존성 및 메타데이터를 정의합니다.

## 개발 노트
*   현재 사용자 목록은 고정된 더미 데이터입니다. 실제 사용자 관리 기능은 구현되어 있지 않습니다.
*   채팅 메시지는 현재 애플리케이션 내에서만 유지되며, 영구 저장 기능은 없습니다.
